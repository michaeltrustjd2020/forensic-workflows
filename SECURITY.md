# Security & Evidence Integrity

- **Admissibility**: Align to CEC §§1271, 1400, 1521.
- **Continuity**: Preserve version history, export method, timestamp, hash, and metadata.
- **Encoding Hygiene**: UTF-8 without BOM; strip smart quotes; normalize newlines.
- **Proof Layers**: Custodian declarations, screenshots, system logs, hash logs.
- **Review**: PRs required to change admissibility or encoding workflows.
