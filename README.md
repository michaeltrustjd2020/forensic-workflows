# Forensic Workflows

**Principles:** Portability • Continuity • Adversarial-proofing • Timestamp integrity • Metadata preservation • Version traceability

## Modules
- prep-layers/: environment isolation, vendor-behavior suppression, preflight checklists
- encoding-checks/: UTF-8 (no BOM), smart-quote stripping, newline normalization
- admissibility-protocols/: CEC §§1271, 1400, 1521 scaffolds
- continuity-crosswalks/: multi-platform evidence mapping (export method, timestamp, hash, metadata)
- dashboards/: status boards, CSV/Excel exports, KPIs
